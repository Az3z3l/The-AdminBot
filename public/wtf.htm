What is an AdminBot?

Adminbot is a way to determine Proof of Concept for client-side web challenges in a CTF.



How does this work?

A client-side challenge in a CTF requires you to steal a cookie or data from an endpoint that is accessible only by an authorized user. This bot employs Puppeteer and Selenium to use the challenge as this authorized user. Then visits the URL you send.



What's with the hashlib stuff?

This service uses hash collision as a Captcha(not exactly a captcha though) to prevent users DoSing the bot. Refer to this for a crude solve script.



What do I send to the bot?

The bot accepts URLs with HTTP and HTTPS protocol.



Am I supposed to pwn this website?

Heck No!! But if you do find some vulns in this service, please report it to the CTF orgs.



Do I HAVE to use this UI?

No. You can easily use the APIs to communicate with the bot.
